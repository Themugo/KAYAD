# KAYAD Development Guide

KAYAD is a production automotive marketplace for Kenya.

## Mission

Build the most trusted automotive marketplace in Africa.

## Core Principles

Never break existing functionality.

Never rewrite working modules unnecessarily.

Never duplicate code.

Always reuse existing services.

Always preserve API compatibility.

Always maintain responsive layouts.

Always maintain RBAC.

Always maintain security.

Always improve accessibility.

Always optimize performance.

Always test builds after modifications.

Always explain architectural decisions.

Always commit logical changes together.

Always update documentation after major changes.

When uncertain, inspect existing code before implementing anything.

Prefer refactoring over rebuilding.

Follow existing coding conventions.
